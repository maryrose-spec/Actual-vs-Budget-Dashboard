Actual vs Budget Executive Dashboard

This project presents an interactive Actual vs Budget Executive Dashboard built using Power BI to help stakeholders monitor financial performance and track key business metrics.

Key Features
Income Actual vs Budget Analysis
Expense Actual vs Budget Analysis
Variance Percentage Calculation
Dynamic KPI Cards
Progress Donut Charts
Project-Level Financial Performance Tracking
Interactive Year, Month, and Project Filters
Conditional Formatting for Variance Analysis
Dynamic Dashboard Titles
Business Questions Answered
How does actual income compare to the budget?
How does actual expense compare to the budget?
Which projects are performing above or below budget?
What percentage of the budget has been achieved?
What are the yearly income and expense trends?
Tools Used
Power BI
DAX
Power Query
Data Modeling
Data Visualization
